日常介紹

松林啟航 部落飛翔

部落簡介

松林部落位於仁愛鄉親愛村北端，臨濁水溪，北有關頭山(1536公尺)以及對萬山(1524公尺)， 南臨法治村，東接花蓮縣，北接春陽村與南豐村，西與聞名中外的「曲冰遺址」所在地萬豐村毗連。 海拔約950至1400公尺間，山系錯綜複雜，蘊含豐富動植物。動植物種屬繁多，又鄰近奧萬大森林遊樂區， 其楓樹讓本區豐富的植物相更添風韻。濁水溪流經本區，溪水沖刷侵蝕致河谷景觀雄偉發達。居民多數以農為業， 主要農作物有茶葉、香糯米、玉米及李、梅、蕃茄等水果，並以栽培香菇為副業，而香糯米為松林部落最具代表性農特產品之一，獨步全台了。

部落傳說

相傳以前松林部落與相鄰的布農族格格不入，超越對方的領地是犯忌的，賽德克族巫師用茅草綁在領地下蠱，當對方超越領地回去後會拉肚子甚或死亡。有的巫師只會放蠱，不會收蠱，造成後代下場境遇欠佳。而擔任治療的巫師後代比較沒有影響，有好有壞。通常孩子失蹤或牛不見了，也會請巫師做法代尋，其他宗教(如基督教)進駐部落之後，巫師下蠱或法術治療等行為都被禁止。 國民政府時代有學校的聯合運動會及教會的聯合禮拜，松林部落開始與南豐村(布農族)接觸。過去通婚犯族群的忌，與外族通婚後就不能回來部落。後來因為部落大多都是親戚，為免近親繁衍，現在已經可以和外族（尤其布農族）通婚 。

農特產

  • 香糯米 五月播種、七月插秧，八月前插秧完畢，如果超過時間結穗的時會有寒流，穗會不飽合，大約十一月中、十二月收成。收成後給農會處理、產銷班、朋友、社區發展協會。而泉水有限，地方想多種一些香糯米，也沒有辦法，所以產量不大價錢較高，香糯米不是部落主要食品、當部落的小孩子回來的時候，飯鍋會放一些香糯米，飯煮好後會特別的香，孩子都特別喜歡吃。
  • 手工藝 部落目前有人在做，但是沒有真正在推行，目前有竹編、織布、搗米器具。竹編目前是何廣榮先生在運作，於親愛國小教國小學生編製，當社區活動時會幫忙推銷，而平常有人訂作才會製作，比較常見的是背籃女生用的叫(薄吻)，男生用的叫(逗肯)。
  • 烏龍茶、段木香菇、黃肉李 、紅肉李 香菇、香糯米、水蜜桃、甜柿、柿加、楊梅、梅子、紅肉李、黃肉李、高接梨。鳥龍茶、新品種茶樹比較少，新品種改良後較好賣，茶業品質春茶、冬茶比較好，秋茶還好1000-1200、因茶葉成本高550以上，夏茶不收成給教會自行去義賣。松林大約有6家茶廠。
