網站理念

松林啟航 部落飛翔


理念圖


為了讓更多人了解仁愛鄉松林部落的原住民，網站的製作團隊希望藉由此網站來讓任何一位進入此網站的使用者都能藉由此網站來了解到他們的文化、生活及需求。


理念圖說明

提供資源：

     藉由此網站使民眾有個平台能了解仁愛鄉松林部落的各項文化及需求，促使民眾願意提供些許幫助。

教　　育：

     獲取資源後，提供閱讀交流空間給部落居民，提高教育水平。

提升能力：

     提高教育後便會讓部落內的學生們得以精進所熱愛的項目。

賺取金錢：

     擁有知識及能力的族人便能在社會中得到更好的工作條件，提高經濟收入。

回饋部落：

     提高經濟收入後便能回饋部落，促進部落發展。
